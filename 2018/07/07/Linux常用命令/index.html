Linux常用命令

Linux常用命令

标签 ： Linux命令

1.ls
-l 文件属性，权限等
-a 所有文件（包括隐藏）
-lh 容量

2.cat
用于查看文本文件

3.pwd
查看当前所在目录

4.cd
cd /xxx/xxxx
cd ./xxx到当前目录的xxx
cd ../xxx到上级目录的xxx

5.grep
用于查找信息，若存在需要的信息就将该行显示，通常与管道命令一起使用。
grep ‘要查找的字符串’ filename

6.find
find / -name cron
命令 目录 表示查找的类型 通配符

7.cp
拷贝文件
-a 将文件特性一起复制（比如权限等等）
-r 递归复制，复制目录以及子目录

8.mv
移动文件
mv file1 file2 file3 dir移动
mv file1 file2重命名

> filename 创建文件,
也可以touch filename

9.rm
删除文件
rm -f强制删除
rm -r递归删除

10.ps
将某个时间点的进程运行情况选取下来并输出
ps aux 查看系统所有的进程数据

此外top也可以查看进程
区别：
ps查看的是自进程创建开始的情况，是区间平均值
top是自上次刷新计算，几乎是实时的

11.kill
杀死进程
kill -9 PID

杀死子进程
kill -ef|grep httpd

killall httpd
杀死进程组内所有进程

12.free
回收内存

13.tar
解压压缩
tar czf demo.tgz filename or foldername
打包
tar xzf demo.tgz
解压
tar tzf demo.tgz
查看压缩文件内的内容

14.chmod
修改访问权限，包括r,w,x
三种用户，所有者，同组用户，其他用户
